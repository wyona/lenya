

  Enterprise Resource Planning
  ----------------------------

  Requirements
  ------------

    JDK-1.4.2
    Apache-Ant-1.6.2
    Apache-Maven-1.0.2

  Getting Started
  ---------------

    Download Libraries: maven
    Compile:            ant jar

  Run
  ---

    Usage:    java -jar build/lib/erp.jar
    Add Task: java -jar build/lib/erp.jar --add-task
