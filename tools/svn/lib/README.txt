This is a README.txt file for JavaSVN - pure Java Subversion client library.

1. DESCRIPTION

JavaSVN is a pure Java Subversion client library.

3. FEATURES

JavaSVN allows to work with Subversion repositories and Subversion working copies. JavaSVN features
direct repository access as well as support for all high level Subversion operation that are 
available with the command line Subversion client. 

This version of JavaSVN supports all Subversion 1.2 features and supports older Subversion repositories.
